# Before You Submit

This is a starting template covering every requirement in the assignment brief.
To meet the academic integrity requirement ("all work submitted is your own"),
personalize and understand each part before handing it in. Go through the
code and make sure you could explain any line if asked.

## Checklist

- [ ] Replace "Jane Student" everywhere in `index.html` with your real name
- [ ] Replace the admission number, course, year, and institution
- [ ] Add your real profile photo to `images/` (see the note in that folder)
      and update the `src` in the `<img>` tag if you rename the file
- [ ] Rewrite the biography in your own words (150–200 words)
- [ ] Update Educational Background with your real schools and dates
- [ ] Update Skills & Hobbies with your actual skills/interests
- [ ] Replace the 5 course units with your real units, codes, and lecturers
- [ ] Test the site by opening `index.html` in Chrome AND Edge (or Firefox)
- [ ] Click every nav link to confirm it scrolls to the right section
- [ ] Click the welcome button and test the contact form's submit/reset
- [ ] Rename the whole folder / zip it as `AdmissionNumber_WebsiteAssignment.zip`

## How the files fit together

- `index.html` — page structure and content (all sections from the brief)
- `style.css` — all visual styling (colors, fonts, layout, responsiveness)
- `script.js` — the welcome button and contact form interactivity
- `images/` — put your profile photo here

Open `index.html` directly in a browser to preview — no server needed.
